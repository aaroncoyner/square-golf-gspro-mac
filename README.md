# Square Golf + GSPro on Mac

This repository contains an unofficial setup guide for connecting a **Square Golf launch monitor** to **GSPro on macOS** using **CrossOver** and GSPro's **OpenAPI/Open Connect** interface.

Read the hosted guide here:

```text
https://aaroncoyner.github.io/square-golf-gspro-mac/
```

## What this repo is

- A user-friendly install guide for Mac users to get Square Golf launch monitors up and running with GSPro.
- A place to keep notes about known-good versions.
- An optional backup/mirror location for open-source SquareGolf Connector release files.

## What this repo is not

This is not the SquareGolf Connector project. The connector is maintained separately by Brent Yates:

```text
https://github.com/brentyates/squaregolf-connector
```

This repository is not affiliated with SquareGolf, GSPro, CodeWeavers/CrossOver, or Brent Yates.


## Backup policy

Use the original SquareGolf Connector repository first. Backup files in this repo, if provided, are intended only as a convenience if the original repository or a specific release becomes unavailable.
